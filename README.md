# FPT SHOP Website Using REACTJS

This project is an e-commerce website developed using ReactJS. It provides a platform where users can buy phones.


## Contributor
- Đàm Quang Chiến
- Đào Quang Khải
## Technologies Used

The project utilizes the following technologies and tools:

- ReactJS
- React Bootstrap
- Bootstrap 4
- React-route-dom
- Json Server Fake
